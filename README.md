# M7011E-Dynamic-Web-Systems


## Dependencies:
- Python 3.12.0 
- Flask

## Setup environment

Clone the repository (using ssh)

	git clone git@github.com:lindstromemil/M7011E-Project.git

In repository folder run these commands

    pip install virtualenv
    python -m venv env
    env\Scripts\activate

When activated

    pip install flask
